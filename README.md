# vyper-cli-hello-world

## Description
A POC for vyper programming language.

## Tech stack
- smart contract

## Docker stack
- vyperlang/vyper

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
